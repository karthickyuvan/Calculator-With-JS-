Description:
A beautifully styled, interactive calculator built with HTML, CSS, and JavaScript. 
This project includes a responsive design, user-friendly interface, and essential arithmetic functionality. 
Features include clear display, operator replacement, backspace for corrections, and accurate calculations.


Key Features:

Performs basic arithmetic operations: addition, subtraction, multiplication, and division.
Handles decimal numbers and ensures proper formatting.
Backspace functionality to delete the last entered character.
Replaces operators if a new one is clicked consecutively.
Clear and responsive design with interactive buttons for enhanced usability.
Gradient background and hover effects for a visually appealing experience.


How to Use:

Clone or download the repository.
Open the index.html file in any web browser.
Interact with the calculator by clicking buttons to perform operations.
